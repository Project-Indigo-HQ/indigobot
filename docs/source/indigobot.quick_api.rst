indigobot.quick_api module
==========================

This module provides a FastAPI implementation for the IndigoBot system, enabling HTTP-based interaction with the bot.

Key Components:
---------------
* API endpoints for sending and receiving messages
* Integration with Chatwoot for customer support
* State management for conversation context
* Query response handling
