indigobot.places_tool module
============================

This module implements the Places Lookup Tool, which integrates with Google Places API to provide location-based information.

The PlacesLookupTool class:
---------------------------
* Retrieves detailed information about places from Google Places API
* Formats opening hours, contact information, and other place details
* Provides a clean interface for place information lookup
