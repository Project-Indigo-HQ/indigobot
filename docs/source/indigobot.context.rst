indigobot.context module
========================

This module provides context management for the IndigoBot system, including place information lookup and integration with the RAG system.

Key Functions:
--------------
* ``lookup_place_info``: Retrieves information about places from external APIs
* ``extract_place_name``: Extracts place names from user input
* ``store_place_info_in_vectorstore``: Stores place information in the vector database
* ``create_place_info_response``: Creates formatted responses with place information
* ``invoke_indybot``: Main function to invoke the IndigoBot with user input
